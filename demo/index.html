jQuery ijedTiles

A simple and configurable tile of images with a box next to it for the title or/and description of the image.

  1. Make sure you have links to jQuery and jquery ijedTiles in your HTML page:

    <script src="/path/to/jquery.js"></script>
    <script> src="/path/to/jquery.ijedtiles.js"></script>
  2. Create an empty div and give it an ID e.g. ijed-box:

    <div id="ijed-box"></div>
  3. Add a <script></script> tag. Use jQuery to select the element and call ijedTiles() on it. Don't forget the jQuery statement that makes sure the document has been fully loaded before running your script. Your code should look like this:

    <script>
     $(function() {
      $('#ijed-box').ijedTiles();
     });
    </script>

That's all you have to do to see ijedTiles in action. The script comes with default options including images and you should see something like below. Hover over the images to see the effect.
